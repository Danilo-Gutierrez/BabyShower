# BabyShower

Sistema web personal para gestionar reservas y regalos del Baby Shower de Thianna Amira.

## Tecnologías usadas

- Python
- FastAPI
- SQLite
- HTML
- CSS
- JavaScript
- Jinja2

## Funcionalidades actuales

- Lista de regalos.
- Reservar regalos.
- Evita reservas duplicadas.
- Panel de administración.
- Contador de regalos disponibles y reservados.
- Confirmación visual al reservar.
- Base de datos local SQLite.
